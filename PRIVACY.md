# Asli Privacy Policy

**Effective Date: March 7, 2026**

Your privacy is the core of **Asli**. This extension is built to be
"Privacy-First" &mdash; No user data is collected, needed, or stored.

### 1. Data Collection & Usage

**Asli** does not collect any personally identifiable information (PII).

- **Browsing History**: Browsing history is not collected, stored, or shared.
  While the extension reads the URL of the active tab to verify if it is on the
  safety list, this verification happens **entirely on the local browser**. No
  URLs are ever transmitted to any external server.
- **User Action Tracking**: Clicks, mouse movements, or keystrokes are not
  tracked.
- **Permissions**: The "tabs" permission is used only to compare the current
  website domain against a locally stored list of official and malicious
  domains.

### 2. Data Sharing

Data is not sold, traded, or shared with third parties. There is no user data to
share.

### 3. Changes to this Policy

If the extension's functionality changes in a way that impacts privacy, this
policy will be updated and users will be notified through an extension update.

### 4. Contact

For any questions about this policy, please open an issue through the GitHub
repository: [github.com/subrat-lima/asli](https://github.com/subrat-lima/asli)
